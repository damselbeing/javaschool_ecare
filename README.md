**ABOUT**

This web application is created for a mobile operator, its main functions are: to manage the contracts, tariffs, options and clients' profiles. Administrator has got privileges to create a new contract, edit and block customersâ€™ contracts, add and change tariffs with options. Regular users have a right to manage their personal contract, inspect options and tariffs. For more project details please read Technical Solution Description file (TSD.pdf).
